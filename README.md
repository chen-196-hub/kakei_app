# README

## [KakeiNet](https://kakei-net.herokuapp.com/)

## アプリケーションの概要

このアプルは家計の計算、TODOリスト、伝言板などの機能を持つウェブサービスです。

## アプリケーションの機能一覧
- 登録機能
  - 新規登録
  - ログイン
  - ログアウト
- Todo list機能
  - 投稿機能
  - 完了チェック”✔︎”機能（Done listに写す）
  - 編集機能
  - 削除機能
- Done list機能
  - 戻す機能（掲示をTodo listに返す）
  - 削除機能
- 家計機能
  - 支出、収入計算機能
- グループ
  - グループ一覧
  - 新規グループ作成機能
